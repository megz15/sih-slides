Problem Statement

Addressing the challenge of integrating Building Integrated Photovoltaic (BIPV) potential estimation and shadow analysis using innovative machine learning and 3D modeling techniques.

ML Model: YOLOv8 + SAM

  • Object detection: YOLOv8
  • Image Segmentation: Segment Anything Model
  • Dataset: xView, Cartosat-3 images (LabelMe)

3D Modelling

  • OpenStreetMap buildings data
  • Calculating extrusion
  • Three.js for rendering

Shadow Calculation

  • pybdshadow for shadow geometry
  • Parameters: sun position, building height, location and time

Solar Energy and PV Output

  • Data from the Global Solar Atlas
  • PVOUT calculations using Solargis algorithms
  • Inputs: GTI, TEMP
  • PV power production potential estimation

Web Application
